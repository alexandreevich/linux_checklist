# Настройка iTerm2 для ускорения работы

## Базовое редактирование строки
- **Ctrl + w** — удалить слово слева
- **Ctrl + u** — удалить всё слева от курсора
- **Ctrl + k** — удалить всё справа от курсора
- **Ctrl + a** — перейти в начало строки
- **Ctrl + e** — перейти в конец строки

## Перемещение по словам 

- **Esc + b** — перейти на слово влево
- **Esc + f** — перейти на слово вправо

## iTerm2 донастройка 
## Дополнительные биндинги (опционально)
> Настраиваются в `Profiles → Keys → Key Mappings`

- **Option + Backspace** — удалить слово  
- **Option + ← / →** — перемещение по словам  
  - Action: `Send Escape Sequence`
  - Esc+: `b` / `f`

## Работа с историей и экраном
- **Ctrl + r** — поиск по истории команд
- **Esc + .** — вставить последний аргумент предыдущей команды
- **Cmd + k** — очистить экран
